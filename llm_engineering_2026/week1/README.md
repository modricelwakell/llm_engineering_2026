# 📦 Week 1 – LLM Engineering Fundamentals (2026)

## 🎯 Overview
This week focuses on building a strong foundation in Large Language Models (LLMs), understanding how to interact with them using APIs, and learning how to process real-world data like website content to build simple AI pipelines.

The main goal is not just using LLMs, but understanding how they work, how to control them, and how to integrate them into small systems.

---

# 📁 Week Structure

## 🔹 Day 1 – First Contact with LLMs
- Calling LLM models using API (`client.chat.completions.create`)
- Understanding request/response flow
- Working with real website content
- Using LLMs for summarization
- Improving output formatting and display

---

## 🔹 Day 2 – API Deep Understanding
- Using OpenAI SDK for model calls
- Making raw API requests without SDK
- Understanding API structure
- Exploring multi-provider usage (OpenAI / Gemini-like models)
- Learning abstraction between models and APIs

---

## 🔹 Day 3 – Tokenization & Model Thinking
- Understanding tokenization process
- Encoding and decoding text into tokens
- Context window limitations
- Memory illusion in LLMs
- How models actually “see” text

---

## 🔹 Day 4 – Website Intelligence & Brochure System (Main Project)
- Building an end-to-end AI system that generates a company brochure from a URL
- Using LLMs to filter relevant website links
- Applying one-shot prompting for link selection
- Fetching and combining multiple webpage contents
- Structuring output into a full brochure including:
  - Company name
  - Main URL
  - Relevant pages and their content
- Improving output using display functions for better formatting
- Understanding how multiple functions work together in a pipeline

---

# 🧠 Key Concepts Learned
- LLM API interaction
- Prompt engineering (system vs user prompts)
- One-shot prompting
- Tokenization basics
- Context window limitations
- Web scraping + LLM integration
- Multi-step AI pipelines
- Function composition
- Output structuring (JSON + Markdown)

---

# 🚀 Final Outcome
By the end of Week 1, I can:
- Interact with LLMs using APIs
- Design effective prompts
- Process and extract information from websites
- Filter relevant data using LLMs
- Build simple AI pipelines that transform raw data into structured outputs
