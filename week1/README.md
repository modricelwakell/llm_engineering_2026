# 📦 Week 1 – LLM Engineering Fundamentals (2026)

## 🎯 Overview

This week focuses on building a strong foundation in Large Language Models (LLMs), understanding how to interact with them using APIs, and learning how to integrate multiple models into simple AI systems.

The goal is not just calling models, but understanding how they work, how to control them, and how to build structured workflows around them.

---

# 📁 Week Structure

## 🔹 Day 1 – First Contact with LLMs & Multi-Model Interaction

* Calling LLMs using API (`client.chat.completions.create`)
* Understanding request/response flow
* Using different providers through OpenRouter
* Introduction to abstraction using LiteLLM
* Building a multi-model conversation system (GPT vs Claude)
* Using system prompts to control model behavior
* Managing conversation history between models

---

## 🔹 Day 2 – API Deep Understanding

* Using OpenAI SDK for model interaction
* Making raw API calls without SDK
* Understanding API structure and parameters
* Exploring multi-provider usage (OpenAI / Gemini-like models)
* Understanding abstraction layers (SDK vs Router vs Framework)

---

## 🔹 Day 3 – Tokenization & Model Thinking

* Understanding tokenization process
* Encoding and decoding text into tokens
* Context window limitations
* Understanding the “memory illusion” in LLMs
* How models interpret and process text internally

---

## 🔹 Day 4 – Website Intelligence & Brochure System (Main Project)

* Building an end-to-end AI system that generates a company brochure from a URL
* Using LLMs to filter relevant website links
* Applying one-shot prompting for link selection
* Fetching and combining multiple webpage contents
* Structuring output into a full brochure including:

  * Company name
  * Main URL
  * Relevant pages and their content
* Improving output formatting using Markdown display
* Understanding how multiple functions work together in a pipeline

---

# 🧠 Key Concepts Learned

* LLM API interaction
* Prompt engineering (system vs user prompts)
* One-shot prompting
* Tokenization basics
* Context window limitations

* Abstraction layers (SDK / Router / Framework)
